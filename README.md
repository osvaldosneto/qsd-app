# qsd-app

## Instalação

Necessário instalação gem sqlite3

### Clone o repositório

```shell
git clone https://github.com/osvaldosneto/qsd-app.git
cd project
```

### Rode o script para alimentar e criar o banco de dados

```shell
ruby bin/setup
```

### Rode o script principal para iniciar a aplicação

```shell
ruby lib/study_diary.rb
```
